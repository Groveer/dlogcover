# DLogCover
A lightweight AST-based tool for statically analyzing log coverage in C++ projects.

## 简介
DLogCover 是一个基于抽象语法树（AST）的轻量级工具，用于静态分析C++项目中的日志覆盖情况。它可以帮助开发人员评估代码中的日志覆盖率，识别没有适当日志记录的代码区域，从而提高代码的可维护性和问题定位能力。

## 主要功能
- 分析函数级日志覆盖率
- 分析分支级日志覆盖率
- 分析异常处理级日志覆盖率
- 识别关键路径的日志覆盖情况
- 生成日志覆盖率报告
- 提供改进建议
